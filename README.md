# Trackify
React library that allows you to get analytics of user's experience

## Installing

1. Step 1:

Install trackify
```bash
npm i trackify
```

2. Step 2:

Just cover Trackify around your app like this
```ts
import { TrackifyProvider } from "trackify";

return(
    <TrackifyProvider>
        <App />
    </TrackifyProvider>
);
```
